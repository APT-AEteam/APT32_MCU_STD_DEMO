# 爱普特STD例程库
# 概述
STD例程库是爱普特AE团队针对爱普特不同系列芯片开发的。用户可以直接在该库上进行二次开发。

# 说明
本仓库是爱普特芯片STD例程的入口，所有的STD例程都可以在此找到。目前提供STD代码的芯片有APT32F102X ,APT32F110X , APT32F171X。更多芯片及其STD代码持续更新中，敬请期待~
